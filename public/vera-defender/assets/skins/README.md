# Vera Defender Skin Assets

This directory contains pixel art skin assets for the game.

## Directory Structure

```
assets/skins/
├── default/          # Default game sprites
│   ├── player_ship.png
│   ├── enemy.png
│   ├── boss.png
│   └── powerup.png
├── ships/            # Player ship skins
│   ├── training_jet.png
│   ├── ace_interceptor.png
│   ├── gundam_custom.png
│   ├── wing_zero.png
│   └── unicorn_destroy.png
├── enemies/          # Enemy skins
│   └── carbon_ace.png
├── bosses/           # Boss skins
│   └── phoenix_immortal.png
└── powerups/         # Powerup skins
    └── plasma_core.png
```

## Asset Specifications

### Sprite Sheets
- Format: PNG with transparency
- Style: Retro pixel art (8-bit/16-bit aesthetic)
- Color Palette: Limited retro palette with neon accents

### Dimensions
- Player Ships: 40x40 to 64x64 pixels
- Enemies: 30x30 to 32x32 pixels
- Bosses: 120x80 to 128x96 pixels
- Powerups: 25x25 to 28x28 pixels

### Animation Frames
- Player Ships: 2-12 frames depending on tier
- Enemies: 2-4 frames
- Bosses: 4-8 frames
- Powerups: 4-6 frames (rotating/pulsing)

## Art Style Guidelines

### Retro Pixel Anime Style
- Sharp pixel edges (no anti-aliasing)
- Limited color palette per sprite (4-8 colors)
- Bold outlines (1-2 pixels)
- Anime-inspired designs with 90s retro aesthetic
- Glow effects using bright center colors with darker edges

### Rarity Color Coding
- Common: Gray/Silver tones
- Rare: Blue accents
- Epic: Purple accents
- Legendary: Gold/Orange accents
- Mythic: Pink/Magenta accents with animated glow

## Preview Images
- 200x200 pixel preview thumbnails
- Centered on transparent background
- Showcase most distinctive features

## Animation Guidelines
- Idle animation: subtle bob/pulse
- Boost/thrust: engine glow animation
- Shooting: weapon flash frames
- Powerups: rotation or pulse cycle
