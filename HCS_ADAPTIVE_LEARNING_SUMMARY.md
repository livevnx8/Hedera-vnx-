## 📋 Quick Start Guide

### **1. Initialize HCS Learning**

```bash
# Enable HCS learning in .env
ENABLE_HCS_LEARNING=true
CONVERSATION_MODEL=./models/fine-tuned/vera-conversation
HEDERA_MODEL=./models/fine-tuned/vera-hedera
ENHANCED_MODEL_ROUTING=intelligent

# Run the demo
npx tsx scripts/hcs-learning-demo.ts
```

### **2. Monitor Learning Progress**

```bash
# Check learning analytics
curl http://localhost:8080/api/learning/analytics

# View system status
curl http://localhost:8080/api/learning/status
```

### **3. Trigger Fine-Tuning**

```bash
# Manual fine-tuning trigger
curl -X POST http://localhost:8080/api/learning/fine-tune

# Export learning data
curl http://localhost:8080/api/learning/export
```
