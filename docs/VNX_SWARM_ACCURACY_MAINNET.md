## Methodology

All accuracy metrics are computed from the local SQLite prediction database (`data/fast_predictions.db`) using the following queries:

```sql
-- Overall accuracy
SELECT COUNT(*), SUM(CASE WHEN correct=1 THEN 1 ELSE 0 END)
FROM fast_predictions WHERE correct IS NOT NULL;

-- Last 10 accuracy
SELECT correct FROM fast_predictions
WHERE correct IS NOT NULL
ORDER BY timestamp DESC LIMIT 10;

-- Last 50 accuracy
SELECT correct FROM fast_predictions
WHERE correct IS NOT NULL
ORDER BY timestamp DESC LIMIT 50;

-- Direction breakdown
SELECT direction, COUNT(*), SUM(CASE WHEN correct=1 THEN 1 ELSE 0 END)
FROM fast_predictions WHERE correct IS NOT NULL GROUP BY direction;
```

A prediction is **correct** if the actual price movement matches the predicted direction within the scoring window.
